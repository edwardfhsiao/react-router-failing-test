# react-router-failing-test

#1 clone this repo
```sh
git clone git@github.com:edwardfhsiao/react-router-failing-test.git
```

#2 go to the folder
```sh
cd react-router-failing-test
```

#3 npm i
```sh
npm i
```

#4 start webpack server
```sh
npm run server_dev
```

#5 open up browser go to http://localhost:9000/
